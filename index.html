Methodology

Prediction Metrics

We predict 6 outcome metrics: resolution type (settled vs litigated), disgorgement amount, civil penalty, prejudgment interest, injunction, and officer/director bar.

Evaluation Process

Ground truth is extracted from SEC release text after cases resolve. Models see only the original complaint—no outcome information. Monetary predictions use 10% tolerance; categorical use exact match.
